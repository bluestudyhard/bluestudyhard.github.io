### 位运算的基本操作符

- & 运算符
- 两个位都为 1 取 1，否则取 0
- | 运算符
- 两个位都为 0 时，结果才为 0
- ~ 取反，就是把数都反过来
- &1 的意思可以理解为%2 判断偶数，与 1 取与后，
- ( **>>** ) 左移，表示整个 2 进制往后移动 1 位
- **比如 >>2 == >>0011 = 0001;**
